vim somefilename - this creates a new file called somefilename

j k h l / arrow keys - to navigate through the file using vim

i - lets you insert text wherever the cursor currently is
o - creates a new line under your where your cursor and allows to insert text;

:q! - quits out and disregards all changes to the file
:x - saves and exits out of a file

:/sometext - vim will try to find sometext inside the file
n - will go the next occurence of sometext
N - will go the previous occurance of sometext

:5 - this will go the line 5 in the file

dd - will delete the current file where the cursor is located
5dd - will delete the 5 lines after and including the line where the cursor is

u - this will undo the last change made

shift+g - takes you to the bottom of the file
gg - will take you to the top of the file
